# Manuelle Icon-Kalibrierung

Da die automatische Erkennung zu langsam/ungenau ist, schneiden wir die Icons einfach kurz selber aus. Das ist präziser.

## Anleitung
1.  Mache einen Screenshot von Instagram auf dem Handy (Startseite).
2.  Schneide die folgenden Icons **klein und eng** aus (nur das Symbol, wenig Hintergrund).
3.  Speichere sie als `.png` in diesen Ordner: `assets/templates/`.

## Benötigte Dateien (Dateinamen müssen stimmen!)

### Navigation (Unten)
- `nav_home.png` (Haus)
- `nav_search.png` (Lupe)
- `nav_plus.png` (Plus in der Mitte)
- `nav_reels.png` (Video Symbol)
- `nav_profile.png` (Dein Profilbild klein rechts)

### Aktionen (Unter Posts)
- `action_heart.png` (Herz / Like)
- `action_comment.png` (Sprechblase)
- `action_share.png` (Papierflieger)

### Sonstiges
- `icon_dm.png` (Messenger oben rechts)
- `icon_search_text.png` (Wenn man auf Lupe klickt -> Das Suchfeld oben)

## Wichtig
- Die Bilder müssen vom **exakt selben Handy** stammen (wegen der Auflösung).
- Am besten mit dem Snipping Tool am PC aus einem Screenshot oder direkt vom Handy übertragen.
